You need a Gmail App Password - not your regular password.

  1. Go to myaccount.google.com/security
  2. Make sure 2-Step Verification is ON
  3. Search for App passwords in the search bar and open it
  4. Give it any name e.g. Email Sender and click Create
  5. Copy the 16-character password shown e.g. abcd efgh ijkl mnop

How to run your script:

1. Place the downloaded send_emails.py in the same folder as your Excel file
2. Open that folder, click the address bar, type cmd and press Enter
3. Type py send_emails.py and press Enter
4. Type yes when prompted to confirm

First time only: the script auto-installs any missing packages when it starts.
